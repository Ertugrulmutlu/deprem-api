Deprem-Api

Bilgilendirme !

Bu sitede Deprem-Api'nin ana sitesidir. Altta bulunan kartlar aracılığıyla nasıl kullanılıcağını öğrenebilirsiniz.

Veriler Kandilli rasathanesinden alınmaktadır : Kandilli rasathanesi için

Ticari kullanım için lütfen kandilli rasathanesi ve Boğaziçi üniversitesi ile görüşünüz.

English documentation can be found on github. Our site does not support English for now.

Kullanım

Butona tıklayarak kullanabileceğiniz komutları öğrenebilirsiniz.

Komutlar

  • 500 Depreme bakma = /tum
  • İlk x depremi gösterme = /ilk/x
  • Son x depremi gösterme = /son/x
  • Son depremi gösterme = /son
  • İlk depremi gösterme = /ilk
  • Seçtiğiniz tarihteki depremleri gösterme = /tum/YYYY.MM.DD
  • Seçtiğiniz tarihteki x kadar depremleri gösterme = /tum/YYYY.MM.DD/x
  • Seçtiğiniz tarihler arasındaki depremleri gösterme = /tum/YYYY.MM.DD-YYYY.MM.DD
  • Seçtiğiniz tarihler arasındaki x kadar depremi gösterme = /tum/YYYY.MM.DD-YYYY.MM.DD-x
  • NOT : Dönüşler Json(dict) şeklindedir.

Özellikleri

Butona tıklayarak Api ile neler yapabileceğinizi öğrenebilirsiniz.

Özellikleri

  • Son 500 depremi görebilir.
  • Dilediğiniz ilk x kadar depremi gösterebilir.
  • Dilediğiniz son x kadar depremi gösterebilir.
  • Son depremi gösterebilir.
  • İlk depremi gösterebilir.
  • Seçtiğiniz tarihteki depremleri gösterebilir.
  • Seçtiğiniz tarihte olan x kadar depremi gösterebilir.
  • Seçtiğiniz tarihler arasındaki depremleri gösterebilir.
  • Seçtiğiniz tarihler arasındaki x kadar depremi gösterebilir.
  • NOT: Rasathane sayfasındaki en üstteki veri api için "ilk", Rasathane sayfasındaki en alttaki veri api için "son"dur.
