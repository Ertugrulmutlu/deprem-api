Deprem-Api

Information !

This site is the main site of Deprem-Api. You can learn how to use it through the cards at the bottom.

Data are taken from Kandilli observatory : for Kandilli observatory

For commercial use, please contact Kandilli Observatory and Boğaziçi University.

Usage

You can learn the commands you can use by clicking the button.

Commands

  • Displaying 500 earthquakes = /tum
  • Displaying first x earthquakes = /ilk/x
  • Displaying last x earthquakes = /son/x
  • Displaying the last earthquake = /son
  • Displaying the first earthquake = /ilk
  • Displaying earthquakes on the date you select = /tum/YYYY.MM.DD
  • Display to x number of earthquakes on the date of your choice = /tum/YYYY.MM.DD/x
  • Display earthquakes between the dates you select = /tum/YYYY.MM.DD-YYYY.MM.DD
  • Display x number of earthquakes between the dates you select = /tum/YYYY.MM.DD-YYYY.MM.DD-x
  • NOTE : Returns are Json(dict).

Properties

You can find out what you can do with Api by clicking the button.

Properties

  • You can display 500 earthquakes.
  • You can display first x earthquakes.
  • You can display last x earthquakes.
  • You can display the last earthquake.
  • You can display the first earthquake
  • You can display earthquakes on the date you select
  • You can display to x number of earthquakes on the date of your choice.
  • You can display earthquakes between the dates you select
  • You can display x number of earthquakes between the dates you select
  • NOTE: The top data on the Observatory page is "first" for the api and the bottom data on the Observatory page is "last" for the api.
